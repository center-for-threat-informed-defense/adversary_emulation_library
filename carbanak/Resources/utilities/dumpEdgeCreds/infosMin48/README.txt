infoMins48

Tool based on: https://github.com/djhohnstein/SharpWeb

For the range, .NET 4.8 is required, so it must be installed on your system.

Recommend building with the csc tool because it leaves behind fewer strings in the binary.
	1. Click the Windows/magnifying glass button 
	2. Type 'dev'
	3. Select Developer Command Prompt
	4. Navigate to infoMins48 source code (infoMins48.cs)
	5. Type 'csc -out:infoMins48.exe infoMins48.cs'
